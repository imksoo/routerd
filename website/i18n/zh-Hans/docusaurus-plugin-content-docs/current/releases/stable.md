# 稳定版里程碑

routerd 以 `vYYYYMMDD.HHmm` 格式频繁发布版本，其中经过评估**可供正式环境使用**的版本，会在每个里程碑时选定为「稳定版里程碑」。初次部署时，请使用本页所列的版本。

## 当前推荐版本

| 项目 | 内容 |
| --- | --- |
| 版本 | **v20260522.1334** |
| 定位 | 首个推荐稳定版里程碑 |
| 运行实绩 | 已在正式环境路由器上运行（已确认 BGP 2-way ECMP 与零中断升级） |
| 二进制文件 | 静态链接（`CGO_ENABLED=0`），通过 CI |

## 推荐 v20260522.1334 的理由

- **BGP 控制平面迁移已完成。** routerd 已从 FRR 切换至自有的 `routerd-bgp` 守护进程，由 `routerd-bgp` 直接维护 eBGP peer。
- **已修正 next-hop 改写的问题（#26）。** 即使上游广告第三方 next-hop，也会通过学习到的 peer 地址将路由写入 kernel FIB，并维持 2-way ECMP。
- **具备正式环境运行实绩。** 升级至 1334 在观测上以 0 秒中断完成。
- **以静态二进制文件发布，** 并通过 CI（构建、测试及 Release workflow）。

## 「稳定版」的定义与注意事项

:::warning API 仍为 v1alpha1
「稳定版里程碑」代表**此版本具备正式环境所需的质量**，并**不保证 API（资源 schema）的向下兼容性**。
:::

- routerd 的资源 API 目前为 **v1alpha1**。**版本之间可能包含破坏性变更。**
- 升级时，请勿依赖向下兼容性，应以**配合新 schema 重新撰写配置（YAML）**为前提进行。
- 本项目不提供迁移兼容层。各版本的变更内容请参阅[变更记录（Changelog）](./changelog.md)。
