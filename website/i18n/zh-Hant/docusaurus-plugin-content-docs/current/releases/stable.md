# 穩定版里程碑

routerd 以 `vYYYYMMDD.HHmm` 格式頻繁發布版本，其中經過評估**可供正式環境使用**的版本，會在每個里程碑時選定為「穩定版里程碑」。初次部署時，請使用本頁所列的版本。

## 目前推薦版本

| 項目 | 內容 |
| --- | --- |
| 版本 | **v20260522.1334** |
| 定位 | 首個推薦穩定版里程碑 |
| 運行實績 | 已在正式環境路由器上運行（已確認 BGP 2-way ECMP 與零中斷升級） |
| 二進位檔 | 靜態連結（`CGO_ENABLED=0`），通過 CI |

## 推薦 v20260522.1334 的理由

- **BGP 控制平面遷移已完成。** routerd 已從 FRR 切換至自有的 `routerd-bgp` 常駐程式，由 `routerd-bgp` 直接維護 eBGP peer。
- **已修正 next-hop 改寫的問題（#26）。** 即使上游廣告第三方 next-hop，也會透過學習到的 peer 位址將路由寫入 kernel FIB，並維持 2-way ECMP。
- **具備正式環境運行實績。** 升級至 1334 在觀測上以 0 秒中斷完成。
- **以靜態二進位檔發布，** 並通過 CI（建置、測試及 Release workflow）。

## 「穩定版」的定義與注意事項

:::warning API 仍為 v1alpha1
「穩定版里程碑」代表**此版本具備正式環境所需的品質**，並**不保證 API（資源 schema）的向下相容性**。
:::

- routerd 的資源 API 目前為 **v1alpha1**。**版本之間可能包含破壞性變更。**
- 升級時，請勿依賴向下相容性，應以**配合新 schema 重新撰寫設定（YAML）**為前提進行。
- 本專案不提供遷移相容層。各版本的變更內容請參閱[變更記錄（Changelog）](./changelog.md)。
